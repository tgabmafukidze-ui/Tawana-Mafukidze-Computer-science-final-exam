# Portfolio Website

This repository contains a simple portfolio website for Tawana Mafukidze.

## Structure

- `index.html` — Home page with a short introduction and career interests.
- `about.html` — Background, education, and motivation for computer science.
- `projects.html` — Portfolio projects and technical exercises.
- `contact.html` — GitHub, email, LinkedIn, and downloadable CV.
- `style.css` — Shared styling for the site.
- `assets/` — Static files including images and CV PDF.
- `latex/` — LaTeX source for the CV.
